# Learning Hub Frontend

Production-ready React application with 42 pages and comprehensive UI component library.

## Features

- 42 fully functional pages
- 20+ reusable components
- Toast notifications system
- Form validation hooks
- Error boundaries
- Loading skeletons
- Quiz and assignment components
- Review and rating system
- Complete accessibility support

## Quick Start

```bash
# Install dependencies
npm install

# Setup environment
cp .env.example .env.local

# Start development server
npm run dev
```

## Build

```bash
# Production build
npm run build

# Preview build
npm run preview
```

## Project Structure

```
src/
├── pages/          # 42 application pages
├── components/     # Reusable components
├── contexts/       # React contexts
├── hooks/          # Custom hooks
├── services/       # API services
├── utils/          # Utilities
└── config/         # Configuration
```
