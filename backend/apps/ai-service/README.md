# AI Service

## Overview
AI-powered features including chatbot, content generation, and smart assistance.

## Responsibilities
- AI chatbot for student support
- Content summarization
- Question answering
- Smart course search
- Auto-tagging and categorization
- Personalized learning paths

## API Endpoints
```http
POST   /api/ai/chat                   # AI chatbot
POST   /api/ai/summarize              # Summarize content
POST   /api/ai/question-answer        # Answer questions
POST   /api/ai/generate-quiz          # Generate quiz from content
POST   /api/ai/tag-content            # Auto-tag content
GET    /api/ai/learning-path          # Suggested learning path
```

## AI Models
- GPT for chat and content
- Embeddings for search
- Classification for tagging

## Integration
- OpenAI API / Custom models
- Course Service - Content analysis
